# Dataset

## Overview

This folder contains the datasets used during Week 2 of the Student Participation Analysis project.

## Files

- **Cleaned_Student_Dataset.csv**
  - Dataset obtained after Week 1 data cleaning.

- **Week2_Cleaned_Dataset.csv**
  - Dataset after additional preprocessing and feature engineering performed during Week 2.

## Data Preparation

The dataset was refined by:

- Removing duplicate records
- Handling missing values
- Standardizing column names and formats
- Converting date columns into datetime format
- Creating new analytical features

## Engineered Features

- Signup Year
- Signup Month
- Signup Day
- Signup Hour
- Application Delay
- Opportunity Duration
- Engagement Intensity Score

The processed dataset is ready for Exploratory Data Analysis (EDA) and predictive modeling.
