# Week 2 – Cohort Participation Analysis with Visual Insights

## Project Overview

This project focuses on preparing a student participation dataset for predictive modeling through data cleaning, feature engineering, exploratory data analysis (EDA), and visualization.

The objective is to identify participation trends, engagement behavior, and opportunity performance while generating insights for machine learning.

---

## Objectives

- Clean and standardize the dataset
- Handle missing values
- Engineer meaningful features
- Perform Exploratory Data Analysis (EDA)
- Visualize participation trends
- Detect anomalies
- Generate actionable insights
- Prepare the dataset for predictive modeling

- Participation varies across opportunity categories.
- Engagement intensity is a strong indicator of participation.
- Early signup positively influences participation.
- Correlation analysis reveals relationships between engineered features.
- Outlier detection highlights highly engaged participants.
