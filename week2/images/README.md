# Visualizations

## Overview

This folder contains all charts generated during Week 2 Exploratory Data Analysis.

## Visualizations Included

### Signup Distribution by Opportunity Type

Shows participation across different opportunity categories.

---

### Participation Trend Line Chart

Displays monthly signup trends.

---

### Engagement Intensity Histogram

Illustrates the distribution of engagement scores.

---

### Correlation Heatmap

Shows relationships among numerical variables.

---

### Boxplot of Engagement Intensity

Identifies anomalies using the IQR method.

---

### Status Distribution

Displays student participation status.

## Purpose

These visualizations help identify participation trends, engagement behavior, and patterns useful for predictive modeling.
