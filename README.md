# Student Participation Analysis

## Project Overview

This project analyzes student participation across internships, courses, events, competitions, and engagement programs.

The objective of Week 1 was to understand the dataset, document its variables, perform initial data cleaning, and generate summary statistics that will support future exploratory analysis and predictive modeling.

---

## Objectives

- Understand dataset structure
- Create a data dictionary
- Identify and address data quality issues
- Generate summary statistics
- Establish a foundation for exploratory data analysis

---

## Tools Used

- Microsoft Excel
- Microsoft Word
- PDF Documentation
- GitHub

---

## Deliverables

### Week 1

- Data Understanding Report
- Data Dictionary
- Missing Value Analysis
- Summary Statistics
- Visualizations

---

## Key Findings

- Internships were the most popular opportunity category.
- The majority of participants were from the United States and India.
- Team Allocated and Rejected were the most common participation outcomes.
- Missing values were minimal except in Opportunity Start Date.
