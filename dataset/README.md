# Dataset

## Overview

The original dataset was provided as part of a Data Analytics Internship project focused on student participation and engagement.

## Dataset Availability

The raw dataset is not included in this repository.

This repository contains only:
- Analysis files
- Documentation
- Reports
- Visualizations

## Reason

The dataset may be subject to usage restrictions or ownership policies established by the internship provider.

## Project Scope

The analysis focuses on:

- Student participation trends
- Opportunity categories
- Application outcomes
- Demographic patterns
- Data preparation and exploration

The results and insights derived from the dataset are documented throughout this repository.
