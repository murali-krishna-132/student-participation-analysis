# 📈 Images

## Overview

This folder contains all visualizations generated during Week 3.

These charts help explain model performance, prediction results, and participation trends.

---

## Visualizations

### Feature Importance

Shows which features contribute the most to predicting student participation.

---

### Model Accuracy Comparison

Compares Logistic Regression and Random Forest performance.

---

### Confusion Matrix

Displays how accurately the Random Forest model classifies participation outcomes.

---

### ROC Curve

Illustrates the classification performance of both models across different thresholds.

---

### Participation Probability Heatmap

Shows average predicted participation probabilities across opportunity categories.

---

### Probability Distribution

Visualizes the distribution of predicted participation strength and dropout risk.

---

### Student Participation Categories

Displays the number of students in each predicted participation category.

---

## Purpose

These visualizations provide an intuitive understanding of:

- Model Performance
- Feature Importance
- Prediction Confidence
- Participation Trends
- Opportunity Rankings

They support interpretation and decision-making for future learning programs.
