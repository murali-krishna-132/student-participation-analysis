# 📊 Week 3 – Predictive Modeling for Student Participation

## 📌 Project Overview

This project is the **Week 3 deliverable** of the Student Participation Analytics Internship. The objective is to build a machine learning model that predicts student participation in learning opportunities using historical signup and engagement data.

The project focuses on transforming cleaned data into actionable insights through predictive analytics, model evaluation, probability estimation, and visualization.

---

## 🎯 Objectives

- Prepare the cleaned dataset for machine learning.
- Train and compare classification models.
- Predict participation probabilities for learning opportunities.
- Evaluate model performance using standard metrics.
- Visualize model outputs.
- Interpret findings to support future recommendations.

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

## 🤖 Machine Learning Models

Two classification algorithms were implemented:

### Logistic Regression
- Used as the baseline model.
- Provides an interpretable benchmark for comparison.

### Random Forest Classifier
- Selected as the primary model.
- Captures complex relationships between features.
- Produced the best predictive performance.

---

## 📈 Model Evaluation

The Random Forest model was evaluated using:

- Accuracy
- Precision
- Recall
- F1-Score
- ROC-AUC Score
- Confusion Matrix

These metrics were used to validate the reliability of the predictive model.

---

## 🔍 Prediction Outputs

The trained model generates:

- Predicted Participation Probability
- Predicted Dropout Probability
- Participation Strength Categories
- Opportunity Ranking based on participation likelihood

These predictions help identify opportunities most likely to attract strong student participation.

---

## 📊 Visualizations

The project includes several visualizations to explain the model outputs:

- Feature Importance Chart
- Model Accuracy Comparison
- Confusion Matrix
- ROC Curve
- Participation Probability Heatmap
- Probability Distribution
- Student Participation Category Distribution

---

## 💡 Key Findings

- Random Forest outperformed the baseline Logistic Regression model.
- Feature importance analysis identified the strongest predictors of student participation.
- Participation probabilities successfully ranked learning opportunities.
- Opportunity categories displayed different participation patterns.
- Predictive analytics provides valuable support for data-driven decision-making.
