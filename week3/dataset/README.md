# 📂 Dataset

## Overview

This folder contains the cleaned dataset used for Week 3 machine learning analysis.

The dataset was prepared during **Week 2** through preprocessing and feature engineering.

---

## Dataset

### RIT_Cleaned_Dataset.csv

The dataset includes historical learner participation records and engineered features used for predictive modeling.

---

## Key Features

Examples include:

- Opportunity Category
- Gender
- Country
- Age Group
- Days to Apply
- Apply to Start Gap
- Signup Information
- Engineered Features
- Participation Indicators

---

## Target Variable

**Is_Dropped**

- 0 → Student participated
- 1 → Student dropped or withdrew

---

## Dataset Usage

The dataset is used to:

- Train Machine Learning Models
- Predict Participation Probability
- Estimate Dropout Risk
- Generate Visual Analytics
- Support Predictive Decision-Making

---

## Note

This dataset has been cleaned and preprocessed before model training.

Missing values, categorical variables, and engineered features were prepared during Week 2.
