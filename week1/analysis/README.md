# Analysis

This folder contains the analysis files used during Week 1 of the Student Participation Analysis project.

## Contents

### Week1_Analysis.xlsx

Excel workbook used for:

* Data exploration
* Missing value analysis
* Summary statistics
* Pivot tables
* Chart creation

### Python Analysis Scripts (.py)

Python scripts used to:

* Load and inspect the dataset
* Perform data cleaning
* Analyze participation patterns
* Generate summary statistics
* Support reproducible analysis

## Analysis Tasks Performed

### Data Understanding

* Examined dataset structure
* Reviewed available features
* Identified important variables for analysis

### Data Cleaning

* Checked for missing values
* Reviewed data consistency
* Verified data types

### Exploratory Analysis

* Opportunity category distribution
* Gender distribution
* Participation status distribution
* Country-wise participation analysis

## Tools Used

* Microsoft Excel
* Python

  * Pandas
  * NumPy
  * Matplotlib (if used)

## Objective

The purpose of this analysis was to establish a strong understanding of the dataset and prepare it for future exploratory analysis and predictive modeling activities.
